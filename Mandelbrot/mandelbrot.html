On the left hand side is the Mandelbrot Set and on the right is the Julia Set - they are both fractals!
Each point represents a complex number (with the frame representing an Argand diagram) - and this number is passed into a simple iterative complex-valued function.
The colours in the fractal represent the output of the function, with bands of the same colour indicating the same output value.
The remarkable thing is that although this function is incredibly simple, it produces an incredibly detailed shape (in fact, infinitely detailed!) - go ahead and explore yourself!


Move your mouse to select fractal
WASD - pan frame
IK - zoom in/out
JL rotate
ZXC + optional SHIFT - change colours
123456789 - change modes
R - reset
YH - increase/decrease level of detail (may cause loss in performance if increased too high)
