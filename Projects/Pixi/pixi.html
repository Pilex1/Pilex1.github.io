How this works: we start with a set of grid points representing complex numbers on an Argand diagram. For each point we assign a colour based on the magnitude of the point. Then we pass it through the Mandelbrot function: z_n+1 ==> z_n ^ 2 + c where z_0 = 0, plotting all the points it passes through with that same starting colour. The result is plotted below.
